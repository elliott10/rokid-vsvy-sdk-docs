目录结构

SDK 目录结构

speech

语音识别服务（speech）目录

  • demo ---- 语音识别服务（speech）和语音合成服务（tts）demo 源码

    • demo_tts.cc ---- 语音合成服务（tts）demo 源码
    • demo_speech.cc ---- 语音识别服务（speech）demo 源码

  • include ---- 编译所需的头文件

    • speech.h ---- 语音识别服务（speech）的头文件
    • tts.h ---- 语音合成服务（tts）头文件
    • speech_common.h
    • rkcodec.h --- opus 头文件
  • src ---- 源文件

uws

使用语音识别服务需要用到的 WebSocket 服务器库

  • tests
